<A><angle0><angle0><angle1><angle1><angle2><angle2><angle3><angle3><angle4><angle4><angle5><angle5><angle6><angle6><SUM><SUM>

первым идёт байт - ascii-символ A, 

затем 14 байт углов для каждой сервы (по два байта на одну серву, диапазон от 0 до 270),
	первые 5 - рука снизу вверх, послдение 2 - камера снизу вверх

затем двухбайтовая контрольная сумма всех байт (15 байт) по формуле Флетчер-16

Подробнее про порядок серв:
	0 - вращат. основание руки
	1 - нижнее звено руки
	2 - среднее звено руки
	3 - верхнее звено руки
	4 - схват руки
	5 - вращат. основание камеры
	6 - звено камеры

Размеры:
	первое звено мм
	второе звено мм
	третье звено мм
	четвертое звено мм (до плоскости кончиков схвата в полном сквачении) или мм (до плоскости кончиков схвата в полном открытии)

Углы вращения каждой сервы от 0 до 270 град, кроме схвата - мин. (открыто), макс. (закрыто)
Начально положение руки - 135 град. для всех серв, кроме схавата, у него мин. ()
